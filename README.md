# 🐍 Python Training

Repositório de estudos e exercícios em Python para prática e aprendizado da linguagem.

## 📁 Estrutura do Projeto

```
Python-Training/
├── beecrowd/          # Exercícios da plataforma BeeCrowd
├── Codes/             # Códigos diversos de prática
├── MaratonaProgramacao/  # Exercícios de maratona de programação
├── main.py            # Arquivo principal
└── pseudociencia.pseudo  # Arquivo de estudo
```

## 🚀 Começando

### Pré-requisitos

- Python 3.8+ instalado
- (Opcional) Ambiente virtual `.venv`

### Executando

```bash
# Ativar ambiente virtual (se necessário)
source .venv/bin/activate

# Executar arquivos Python
python nome_do_arquivo.py
```

## 📚 Conteúdo

- **beecrowd/**: Desafios de programação do BeeCrowd
- **MaratonaProgramacao/**: Exercícios para maratonas de programação
- **Codes/**: Exemplos e práticas diversas
